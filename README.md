🖥️ Relatório de Instalação
Linux Mint 22.3 “Zena”
📌 1. Introdução

O Linux Mint é um sistema operacional baseado no Ubuntu, conhecido por sua facilidade de uso, estabilidade e interface amigável.
A versão 22.3 “Zena” traz melhorias de desempenho, segurança e compatibilidade com hardware moderno.

💻 2. Requisitos do Sistema

Para a instalação, são recomendados os seguintes requisitos mínimos:

Processador: 64 bits
Memória RAM: 2 GB (recomendado 4 GB ou mais)
Armazenamento: 20 GB livres
Mídia de instalação: Pendrive de pelo menos 8 GB
Software para boot: Rufus ou balenaEtcher
