O Linux Mint é um sistema operacional baseado no Ubuntu, conhecido por sua facilidade de uso, estabilidade e interface amigável.
A versão 22.3 “Zena” traz melhorias de desempenho, segurança e compatibilidade com hardware moderno.

💻 2. Requisitos do Sistema

Para a instalação, são recomendados os seguintes requisitos mínimos:

Processador: 64 bits
Memória RAM: 2 GB (recomendado 4 GB ou mais)
Armazenamento: 20 GB livres
Mídia de instalação: Pendrive de pelo menos 8 GB
Software para boot: Rufus ou balenaEtcher


⬇️ 3. Download da ISO

O arquivo ISO foi obtido no site oficial do Linux Mint Team.

Passos:

Acessar o site oficial
Escolher a versão (Cinnamon, MATE ou XFCE)
Baixar a ISO do sistema

⚙️ 5. Processo de Instalação
Etapas realizadas:
Reiniciar o computador
Acessar a BIOS/UEFI (teclas como F2, DEL ou F12)
Selecionar o pendrive como dispositivo de boot
Escolher a opção “Start Linux Mint”
Iniciar o instalador clicando em “Install Linux Mint”
Configurações escolhidas:
Idioma: Português (Brasil)
Layout do teclado: ABNT2
Conexão com a internet: Ativada
Tipo de instalação:
( ) Apagar disco e instalar
( ) Instalação junto com outro sistema (dual boot)
Fuso horário: Brasil
Usuário e senha definidos
